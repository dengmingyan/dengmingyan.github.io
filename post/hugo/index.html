Hugo

安装Hugo以及搭建个人博客

简介

官方网站：https://gohugo.io/

Hugo是一个静态网站生成器

1.安装Hugo

在Hugo官方网站下载压缩包后可由SFTP上传至Linux

mkdir hugo

tar -zxf hugo_0.107.0_Linux-64bit.tar.gz -C hugo //解压hugo压缩包至hugo文件夹

cp ./hugo /usr/local/bin

hugo version //查看hugo版本信息

2.创建个人博客

hugo new site danta-blog(你的博客名称) //创建个人博客

3.下载主题

主题网站：完整列表|雨果主题 (gohugo.io)

cd danta-blog

git clone github网站 themes/主题名字

git clone https://github.com/dataCobra/hugo-vitae themes/hugo-vitae //下载主题至themes/hugo-vitae

4.配置主题

找到你下载好的主题配置文件config.toml，将它复制到你的博客配置文件config.toml里

vim config.toml

5.本地启动hugo生成网站

hugo -D //更新配置

hugo server --bind "0.0.0.0" --buildDrafts //启动网站

如上图所示则启动成功，恭喜你已经成功配置好个人博客啦~

在Github部署Hugo

1.新建仓库

2.hugo生成网页，上传至Github

hugo --theme=hugo-vitae --baseURL="https://dengmingyan.github.io/" --buildDrafts

执行语句后会在博客目录下生成一个public文件

再下载应用Xftp 7将public文件复制至本地，然后上传到github仓库

3.访问部署好的网站

https://dengmingyan.github.io/
