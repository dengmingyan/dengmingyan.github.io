在Linux上安装Nginx

安装Nginx

1.安装Nginx

yum install nginx

2.配置防火墙开放端口

system start firewalld

firewalld-cmd --add-server=http --permanent

firewalld-cmd --add-port=1313/tcp --permanent

firewalld-cmd --reload

firewall-cmd --list-ports

3.启动Nginx

systemctl start nginx

hugo server --bind "0.0.0.0" --buildDrafts //启动hugo

4.配置Nginx

vim /etc/nginx/nginx.conf

将user后的nginx改为root

vim /etc/nginx/conf.d/www.conf

写入hugo静态网站内容

5.重启nginx

systemctl restart nginx
