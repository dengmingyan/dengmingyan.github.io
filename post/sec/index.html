在Linux（openeuler）上安装Git

在Linux（openeuler）上安装git

官方下载地址：

Git (git-scm.com)

下载与安装

1.在git官网复制下载地址

2.在Linux上进行下载

wget https://www.kernel.org/pub/software/scm/git/git-2.38.1.tar.gz

3.解压下载好的安装包

tar -zxvf git-2.38.1.tar.gz -C /usr/local

cd /usr/local/git-2.38.1

4.删除旧版本（若无安装过git则无需）

yum -y remove git

5.编译git源码

make prefix=/usr/local/git all

6.安装git

make prefix=/usr/local/git install

7.进行环境变量配置

1.修改/etc/profile

vim /etc/profile

2.在最后一行添加：

export **GIT_HOME**=/usr/local/git

export **PATH**=$PATH:$GIT_HOME/bin

3.输入:wq保存文件
4.刷新/etc/profile文件

source /etc/profile

8.验证git

git --version

如果出现git版本信息，恭喜你成功安装git啦！！

配置git相关信息

1.创建一个gitee仓库

官方网站：Gitee - 基于 Git 的代码托管和研发协作平台

注册成功后创建一个新的gitee仓库

2.在linux上与git绑定（在gitee仓库会有代码提示）

Git全局设置：

git config --global user.name "Dengmingyan"

git config --global user.email "12166137+dengmingyan@user.noreply.gitee.com"

创建Git仓库：

mkdir test
cd test
git init 
touch README.md
git add README.md
git commit -m "first commit"
git remote add origin https://gitee.com/dengmingyan/test.git
git push -u origin "master"

(此处代码因人而异，请勿直接复制)

3.gitee仓库提示代码提交成功
